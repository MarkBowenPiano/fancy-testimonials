Plugin for displaying testimonials via a shortcode for use on posts and pages.

== Description ==

Fancy Testimonials is a plugin for displaying testimonials via a shortcode for use on posts and pages made by Dijitul - the UK <a href=\"http://dijitul.com/wordpress-web-design/\">wordpress web design</a> experts

The plugin comes with very minimal basic styling. Please feel free to change this as you wish. 

There are instructions for the slider part of this plugin in the slider.js file included in the plugin folder. 

To use the shortcodes for the Dijitul Testimonials use the following structure. 

<pre><code>

[testimonial-group]

[testimonial author="Insert Author Name, Company etc here"]

Lorem ipsum dolor sit amet, consectetur adipisicing elit. Non, fugiat, deleniti, mollitia ipsam blanditiis commodi nihil tempora nulla quod rerum temporibus inventore molestias recusandae cum consequatur nobis sapiente tempore quia.

[/testimonial]

[/testimonial-group]

</code></pre>
